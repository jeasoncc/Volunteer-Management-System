# 莲花生命关怀志愿者官网

这是一个为莲花生命关怀志愿者团队创建的官方网站，旨在为临终关怀、重症陪护和家属支持提供温暖、专业的在线展示平台。

## 项目概述

莲花生命关怀项目起源于寺院长期的助念与临终关怀实践，致力于为临终者与家属提供温柔、可持续的陪伴支持。本官网为该项目提供全面的在线信息服务。

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite
- **路由管理**: TanStack Router
- **样式框架**: TailwindCSS
- **UI组件**: 自定义组件库
- **部署方式**: 静态网站部署

## 功能模块

### 核心页面
1. **首页** - 项目介绍和核心服务概览
2. **关于我们** - 组织背景、使命愿景和团队介绍
3. **志愿服务** - 详细服务类型、原则和流程
4. **项目故事** - 志愿服务经历和感人案例分享
5. **新闻动态** - 最新活动和媒体报道
6. **加入我们** - 志愿者要求和加入流程
7. **捐赠支持** - 多种支持方式
8. **联系我们** - 联系方式和在线留言
9. **常见问题** - 常见疑问解答

### 技术特性
- 响应式设计，完美适配各种设备
- SEO优化，提升搜索引擎可见性
- 代码分割和懒加载，优化性能
- 组件化架构，易于维护和扩展
- 无障碍访问支持

## 部署说明

### 构建项目
```bash
npm run build
```

构建完成后，静态文件将生成在 `dist/` 目录中。

### 部署方式

#### Nginx部署
1. 将 `dist/` 目录内容复制到服务器
2. 配置Nginx指向该目录
3. 启用gzip压缩和静态资源缓存

#### Docker部署
```bash
# 构建镜像
docker build -t lotus-care-web .

# 运行容器
docker run -d -p 80:80 lotus-care-web
```

## 维护说明

### 更新内容
直接修改对应页面组件文件即可更新内容。

### 添加新页面
1. 在 `src/pages/` 目录创建新页面组件
2. 在 `src/routes/routeTree.gen.tsx` 中添加路由配置
3. 在 `src/components/Header.tsx` 和 `src/components/Footer.tsx` 中添加导航链接

### 样式定制
所有样式均使用TailwindCSS类名，可在 `src/styles.css` 中进行全局样式调整。
