## 问题排查

### 常见问题

| 问题 | 检查项 | 解决方案 |
|------|--------|----------|
| 页面显示 0 数据 | API 连接 | 检查 `.env` 配置 |
| API 请求失败 | CORS 配置 | 检查 API CORS 设置 |
| 页面刷新 404 | Nginx 配置 | 添加 `try_files` 规则 |
| 静态资源 404 | 文件路径 | 检查 Nginx root 路径 |
| 数据不更新 | 自动刷新 | 检查浏览器控制台 |
