# 项目截图

这个目录用于存放项目的截图和演示图片。

## 📸 需要的截图

请添加以下截图到此目录：

### 1. login.png
- **内容**: 登录页面
- **尺寸**: 1920x1080 或 16:9 比例
- **要求**: 展示美化后的登录界面，包含渐变背景

### 2. volunteer-list.png
- **内容**: 义工列表页面
- **尺寸**: 1920x1080 或 16:9 比例
- **要求**: 展示完整的义工列表，包含搜索、分页等功能

### 3. checkin-report.png
- **内容**: 考勤报表页面
- **尺寸**: 1920x1080 或 16:9 比例
- **要求**: 展示月度考勤报表，包含统计数据和列表

### 4. dashboard.png (可选)
- **内容**: 仪表盘/首页
- **尺寸**: 1920x1080 或 16:9 比例
- **要求**: 展示系统概览和关键指标

### 5. mobile.png (可选)
- **内容**: 移动端界面
- **尺寸**: 375x812 (iPhone X) 或类似比例
- **要求**: 展示响应式设计在移动端的效果

## 📝 截图指南

### 拍摄建议
1. 使用真实但脱敏的数据
2. 确保界面干净整洁
3. 选择合适的浏览器窗口大小
4. 使用高分辨率截图

### 工具推荐
- **macOS**: Cmd + Shift + 4
- **Windows**: Win + Shift + S
- **Chrome DevTools**: 设备模拟器截图
- **在线工具**: [Screely](https://screely.com/) - 添加浏览器边框

### 优化建议
- 压缩图片大小（推荐使用 [TinyPNG](https://tinypng.com/)）
- 保持图片清晰度
- 统一截图风格
