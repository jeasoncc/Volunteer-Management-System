# 考勤详情页面导航指南

## 如何进入考勤详情页面

### 方法 1：从侧边栏进入（推荐）✨

1. **点击侧边栏的"考勤管理"菜单**
   - 位置：左侧导航栏
   - 图标：📋 ClipboardCheck
   - 文字：考勤管理

2. **进入考勤主页面** (`/checkin`)

3. **点击右上角的"原始记录"按钮**
   - 位置：页面右上角
   - 图标：📄 List
   - 文字：原始记录

4. **进入考勤详情页面** (`/checkin/details`)

### 方法 2：直接访问 URL

直接在浏览器地址栏输入：
```
http://localhost:5173/#/checkin/details
```

或者（如果使用生产环境）：
```
https://your-domain.com/#/checkin/details
```

## 页面功能说明

### 考勤详情页面 (`/checkin/details`)

**主要功能**：
- ✅ 查看原始打卡记录
- ✅ 按日期范围筛选
- ✅ 按用户（莲花斋ID）筛选
- ✅ 分页浏览
- ✅ 删除记录
- ✅ 查看详细信息（签到时间、地点、状态等）

**数据来源**：
- 表：`volunteer_checkin`（原始打卡记录表）
- 每次打卡生成一条记录

**适用场景**：
- 查看详细的打卡记录
- 审计考勤数据
- 查找特定时间的打卡记录
- 删除错误的打卡记录

### 考勤主页面 (`/checkin`)

**主要功能**：
- ✅ 查看月度报表（汇总数据）
- ✅ 管理汇总记录
- ✅ 导出考勤统计表
- ✅ 编辑和删除汇总记录

**数据来源**：
- 表：`volunteer_checkin_summary`（考勤汇总表）
- 每人每天一条汇总记录

**适用场景**：
- 查看月度考勤统计
- 导出考勤报表
- 管理汇总数据

## 两个页面的区别

| 特性 | 考勤主页面 | 考勤详情页面 |
|------|-----------|-------------|
| **路由** | `/checkin` | `/checkin/details` |
| **数据表** | `volunteer_checkin_summary` | `volunteer_checkin` |
| **数据粒度** | 每人每天一条 | 每次打卡一条 |
| **主要用途** | 报表统计 | 详细查询 |
| **入口** | 侧边栏直接进入 | 从主页面点击"原始记录" |
| **功能** | 月度报表、导出 | 详细记录、筛选 |

## 常见问题

### Q1: 为什么找不到考勤详情页面？

**A**: 考勤详情页面不在侧边栏菜单中，需要：
1. 先进入"考勤管理"
2. 再点击"原始记录"按钮

### Q2: "原始记录"和"记录管理"有什么区别？

**A**: 
- **原始记录** (`/checkin/details`) - 查看原始打卡记录（每次打卡一条）
- **记录管理** - 考勤主页面的一个视图，管理汇总记录（每人每天一条）

### Q3: 如何查看某个义工的详细打卡记录？

**A**: 
1. 进入考勤详情页面 (`/checkin/details`)
2. 在"莲花斋ID"输入框输入义工的 ID（如 LHZ0001）
3. 点击"查询"按钮

### Q4: 考勤详情页面可以编辑记录吗？

**A**: 
- 当前版本：只能删除记录
- 未来计划：添加编辑功能

### Q5: 如何导出考勤详情数据？

**A**: 
- 考勤详情页面暂不支持导出
- 如需导出，请使用考勤主页面的"导出"功能
- 导出的是汇总数据（每人每天一条）
