## 🆘 常见问题

**Q: 导入时提示重复怎么办？**
A: 说明该义工已存在，可以跳过

**Q: 部分义工导入失败？**
A: 检查数据格式，特别是日期格式（YYYY-MM-DD）

**Q: 如何处理无手机号的义工？**
A: 可以先用临时号码，后续更新

**Q: 重复手机号怎么处理？**
A: 系统应该允许，因为可能是家庭成员共用
