## 🆘 遇到问题？

### 常见问题

**Q: 批量导入功能在哪里？**
A: 义工管理页面 → 点击"批量导入"按钮

**Q: 导入时提示"身份证号已存在"**
A: 该义工已在系统中，可以跳过

**Q: CSV文件格式不对？**
A: 确保使用UTF-8编码，Excel可能需要另存为CSV UTF-8格式

**Q: 如何处理无手机号的义工？**
A: 可以先用临时号码，后续更新

**Q: 部分义工导入失败？**
A: 检查数据格式，特别是日期格式（YYYY-MM-DD）
