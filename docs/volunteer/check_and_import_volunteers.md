## 🎯 快速检查方法

### 方法1：使用API批量检查（推荐）

创建一个检查脚本，批量查询这些义工是否已存在：

```javascript
// 所有需要检查的身份证号
const idNumbers = [
  "440513199709122913", // 林楚立
  "445281198404086317", // 房石安
  "360521197504294819", // 钟礼兵
  // ... 添加所有身份证号
];

// 批量检查
async function checkExistingVolunteers() {
  const existing = [];
  const missing = [];
  
  for (const idNumber of idNumbers) {
    try {
      const response = await fetch(`/api/volunteer?idNumber=${idNumber}`);
      if (response.ok) {
        const data = await response.json();
        if (data.volunteers && data.volunteers.length > 0) {
          existing.push({ idNumber, name: data.volunteers[0].name });
        } else {
          missing.push(idNumber);
        }
      }
    } catch (error) {
      console.error(`检查 ${idNumber} 时出错:`, error);
    }
  }
  
  console.log(`✅ 已存在: ${existing.length} 个`);
  console.log(`❌ 需要添加: ${missing.length} 个`);
  
  return { existing, missing };
}
```
